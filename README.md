# Boilerplate
This is a web frontend boilerplate. It uses:
- Webpack 5
- Pug
- SCSS with Postcss and Autoprefixer
- JS ES2015+ with Babel
- LocomotiveScrollJS

# Setup
- `npm install`
- `npm start`

# CLI
- `webpack serve`
- `webpack --watch`
- `webpack`
