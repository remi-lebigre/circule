# Boilerplate
This is a web frontend boilerplate. It uses:
- Webpack 5
- Pug
- SCSS with Postcss and Autoprefixer
- JS ES2015+ with Babel
- LocomotiveScrollJS
- BarbaJS for transitions
- Vercel conf for CI/CD
- Node script for content fetching from remote

# Setup
- `npm install`
- `npm start`

# CLI
- `webpack serve`
- `webpack --watch`
- `webpack`
